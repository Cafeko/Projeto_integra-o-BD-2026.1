# Pipeline de Dados: Sistema de Acompanhamento de Informações de Mercado para Regulação Econômica
>  Projeto da disciplina de Banco de Dados 2026.1

Este projeto implementa e compara duas arquiteturas fundamentais de Engenharia de Dados — ETL Clássico (Python/Pandas) e ELT Moderno (dbt/SQL) — para processar, integrar e modelar dados públicos do setor elétrico brasileiro, mais a implementação de uma Modelagem Dimensional (Esquema Estrela), transformando dados operacionais brutos em um Data Warehouse otimizado para consultas analíticas e Business Intelligence (BI).

Fonte do Banco de  Dados: Sistema de Acompanhamento de Informações de Mercado para Regulação Econômica (SAMP), disponibilizado pela Agência Nacional de Energia Elétrica (ANEEL).
